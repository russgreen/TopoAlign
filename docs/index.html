TopoAlign

TopoAlign provides a set of Revit add-in commands to align and manage topographic surfaces and slabs. Typical outcomes include adding sampled points to topography from curves or intersections, aligning floor slab shapes to the underlying terrain, and resetting regions of topography based on source/target selections.

System Requirements

  • Autodesk Revit: 2022, 2023, 2024, 2025, or 2026
  • .NET Framework 4.8 for Revit 2022–2024; .NET 8.0 (Windows) for Revit 2025–2026
  • Permissions to install Revit add-ins under your user profile

Installation

Installer

If an installer is provided, run it and start Revit. The add-in will appear in the Add-Ins tab.

Manual

  1. Build or obtain the add-in DLLs and the TopoAlign.addin manifest.
  2. Copy them to: %AppData%\Autodesk\Revit\Addins\<RevitYear>\.
  3. Launch Revit and verify the commands in the TopoAlign tab.

Quick Start

  1. Open a model and switch to a suitable view (3D for most point projection workflows; plan for region reset).
  2. Use Align to Element to add points to topography from selected model elements.
  3. Use Align to Topo to shape a floor slab to the terrain, with an optional vertical offset.
  4. Use Points Along Contours or Points On Surface to add points to topography from selected lines.
  5. Use Points At Intersection to place points where a face intersects the topography.
  6. Use Reset Topo Region to replace points within a rectangle from a source topo into a target topo.

Commands

Align to Element

Dialog-driven alignment of topography to selected elements. Provides a guided workflow.

Align to Topo

Shapes a floor slab to match the terrain within its outline, with optional vertical offset.

Points from Lines

Samples points on the topographic surface beneath selected curves and places them on the topo.

Points Along Contours

Adds points to topography at a fixed interval along selected model lines/curves.

Points At Intersection

Computes and adds points where a selected face intersects the topography.

Reset Topo Region

Replaces points in a rectangular region of a target topo with points from a source topo.

Configuration & Paths

  • Add-in manifest: %AppData%\Autodesk\Revit\Addins\<RevitYear>\TopoAlign.addin
  • Installed binaries: same folder as manifest unless configured otherwise.
  • Settings: stored per-user (Placeholder) and persisted between runs for unit/offset/divide inputs.

Logging

Anonymized usage is logged to Google Analytics. No personal data is collected. The information collection is the addin version, Revit version, command used and the country of the user.
