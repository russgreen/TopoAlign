Align to Topo

Shapes a selected floor slab to match the underlying terrain within its footprint. An optional vertical offset can be applied to maintain clearances or design intent.

Usage / Steps

  1. Launch the command.
  2. Enter a vertical offset (can be negative or zero). Units respect project settings.
  3. Select the floor to align when prompted.
  4. Select the target topography/toposolid.
  5. The slab shape editor is populated with sampled points from the terrain.

Options / Parameters

  • Vertical Offset (double) — Default0; offset applied to the sampled terrain before shaping the slab.
  • Units — Choose from valid length units. Defaults to the project’s length units.

Output / Results

  • The floor’s slab shape is modified to follow the terrain inside the floor boundary.
  • Existing slab shape is reset prior to applying new points.

Tips & Pitfalls

  • Apply small offsets to avoid self-intersections or extreme warping near edges.
  • Ensure the floor outline does not cross subregions or subdivisions that could block sampling.

Troubleshooting

  • No points added → Floor or topo not selected → Rerun and ensure both selections complete.
  • Slab fails to update → Geometry conflicts → Reset slab shape and try a smaller offset.
