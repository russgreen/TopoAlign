Reset Region

Replaces target topography points inside a user-drawn rectangle with points from a source topography filtered by phases.

Usage / Steps

  1. Ensure you are in a plan view.
  2. Choose source/target phases and select corresponding topos.
  3. Draw a rectangle to define the reset region.
  4. The target region is cleared and repopulated from the source.

Demo Video

Options / Parameters

  • Source Phase — Filters source topo candidates.
  • Target Phase — Filters target topo candidates.
  • Selection Rectangle — Defines region bounds in the active view.

Output / Results

  • Target topo points within the rectangle mirror the source topo points.

Tips & Pitfalls

  • Use a scope box or temporary isolate to avoid mis-selections.
  • Ensure both topos overlap the selection rectangle.

Troubleshooting

  • Must be in a plan view → Switch views and retry.
  • No topos listed → Phase filters exclude them → Pick different phases.
