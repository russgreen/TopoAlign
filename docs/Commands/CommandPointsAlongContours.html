Points Along Contours

Adds points to the topographic surface at a fixed spacing along selected model lines or curves. Useful for sampling contours, boundaries, or path-aligned features.

Usage / Steps

  1. Launch the command.
  2. Set the spacing (“Divide”) and optional vertical offset. Choose units.
  3. Select one or more lines/curves when prompted.
  4. Confirm to add points to the topo at regular intervals.

Options / Parameters

  • Divide Distance (double) — Interval between points along input curves.
  • Vertical Offset (double) — Optional Z offset applied to created points.
  • Units — Length units for inputs; defaults to project units.

Output / Results

  • Topo points are created at regular intervals along the selected curves.
  • Duplicate points are removed using a positional tolerance.

Tips & Pitfalls

  • If the selected curves form a closed loop, you may be prompted to clean interior points.
  • Ensure curves are contiguous if you expect an ordered path.

Troubleshooting

  • No points added → Invalid or empty selection → Rerun and select model lines/curves.
  • Uneven spacing → Curves of varying types → Consider simplifying or segmenting inputs.
