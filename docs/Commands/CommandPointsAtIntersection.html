Points At Intersection

Computes intersections between a selected face and the topographic surface, then places points on the topo at those intersections.

Usage / Steps

  1. Switch to a3D view.
  2. Launch the command.
  3. Select the topo when prompted.
  4. Select a face on an element.
  5. Points are computed and placed at the intersection locations.

Demo Video

Options / Parameters

  • Sampling Distance (double) — Placeholder: step used for triangle intersection sampling.
  • Units — Uses project length units when applicable.

Output / Results

  • Topo points are added where the face intersects the terrain.
  • Duplicate points are removed using a geometric tolerance.

Tips & Pitfalls

  • Prefer simple faces for clearer intersections (planar or gently curved).
  • Ensure the face and topo overlap in3D space.

Troubleshooting

  • Must be in a3D view → Switch to a3D view and retry.
  • No intersections found → Elements do not intersect → Adjust selection or view.
