Points from Lines

Samples the topographic surface beneath selected lines/curves and creates points on the topo at a fixed spacing along those curves.

Usage / Steps

  1. Launch the command.
  2. Set the spacing (“Divide”). Vertical offset is disabled for this command.
  3. Select the target topo, then select one or more lines/curves.
  4. Points are projected onto the topo along the selected paths.

Demo Video

Options / Parameters

  • Divide Distance (double) — Interval between sample points along the curves.
  • Units — Length units for inputs; defaults to project units.

Output / Results

  • Topo points are added at sampled positions, with duplicates removed using a tolerance.

Tips & Pitfalls

  • Ensure curves are connected if a continuous path is required.
  • Consider isolating the topo and inputs to simplify selection.

Troubleshooting

  • “You must be in a3D view” → Switch to a3D view and retry.
  • Empty result → Curves may be out of topo range → Check overlaps and spacing.
